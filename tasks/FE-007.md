# FE-007: Export/Import UI

**Статус:** Выполнено
**Дата:** 2026-02-25

## Описание

Реализован UI экспорта и импорта состояния приложения в формате JSON на главной странице.

## Что сделано

1. **Компонент `ExportImportSection`** (`frontend/src/components/ExportImportSection.tsx`):
   - Кнопка «Экспорт»: вызывает `GET /api/v1/export`, формирует Blob и скачивает JSON-файл с timestamp в имени.
   - Кнопка «Импорт»: открывает file picker (`.json`), парсит файл, валидирует JSON на клиенте, отправляет `POST /api/v1/import`.
   - После успешного импорта обновляется список типов и показывается success-сообщение.
   - Ошибки (невалидный JSON, ошибки backend) отображаются локализованно.

2. **Интеграция в `MainPage`** (`frontend/src/pages/MainPage.tsx`):
   - `ExportImportSection` размещён внизу страницы с визуальным разделителем.
   - Callback `onImported` вызывает перезагрузку списка типов.

3. **CSS** (`frontend/src/styles.css`):
   - Добавлены стили `.export-import-section`, `.export-import-actions`, `.success-message`.
